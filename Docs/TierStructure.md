# Tier Structure

## Overview

Hyperclocked Factory has **4 tiers** for v1.0. Each tier unlocks when the player produces a specific key material, gating complexity and new systems behind tangible factory milestones. The final tier (HV) rewards the player with access to **Applied Energistics** — an advanced automated item logistics system.

```
[STEAM] ──── Bronze Age ────► Brick Furnace ──► Steel
                                                   │
                                                [LV] ──── Electric Age ────► Blast Furnace ──► Aluminium
                                                                                                   │
                                                                                                [MV] ──► Stainless Steel
                                                                                                              │
                                                                                                           [HV] ──► Applied Energistics
```

---

## Tier 0 — Steam

**Unlock condition:** Available from the start.

**Theme:** The Bronze Age. No electricity. Machines run on steam generated by burning coal. The player's first goal is to set up ore extraction and produce Bronze.

### Key Materials
| Material | Recipe | Machine |
|---|---|---|
| Copper Ingot | Copper Ore → smelt | Steam Furnace |
| Tin Ingot | Tin Ore → smelt | Steam Furnace |
| Copper Dust | Copper Ore → macerate | Steam Macerator |
| Tin Dust | Tin Ore → macerate | Steam Macerator |
| Bronze Ingot | Copper Dust + Tin Dust (3:1) → alloy | Steam Alloy Smelter |
| Bronze Plate | Bronze Ingot → compress | Steam Compressor |

### Resource Nodes Available
- Copper Ore
- Tin Ore
- Coal

### Machines
| Machine | Function | Power |
|---|---|---|
| Steam Extractor | Pulls ore from node into pipe network | — |
| Steam Boiler | Coal + Water → Steam | Produces steam |
| Steam Furnace | Ore/Dust → Ingot | Consumes steam |
| Steam Macerator | Ore → 2× Dust (ore doubling) | Consumes steam |
| Steam Alloy Smelter | 2× inputs → Alloy Ingot | Consumes steam |
| Steam Compressor | Ingot → Plate | Consumes steam |
| Brick Furnace | Iron + Coal Coke → Steel *(unlock trigger)* | No power — slow burn |

> **Note:** The Brick Furnace is a special primitive machine. It requires no steam or electricity but is very slow. It exists solely as the bridge between Steam and LV tier. It is unlocked via the final Steam research node.

### Recipe Depth
**2–3 steps** — e.g. Ore → Dust → Alloy → Plate

### Tier Unlock Trigger
Produce **1× Steel Ingot** in the Brick Furnace. This action unlocks the LV tier research branch.

## Tier 1 — LV (Low Voltage)

**Unlock condition:** Produce Steel in the Brick Furnace.

**Theme:** The Electric Age. Machines are now powered by EU (energy units) from coal-powered generators. Steel enables stronger machines and opens the path to more complex alloys. The player must set up a power cable network for the first time.

### Key Materials
| Material | Recipe | Machine |
|---|---|---|
| Steel Ingot | Iron Dust + Coal Coke → smelt | Brick Furnace / LV Blast Furnace |
| Steel Plate | Steel Ingot → compress | LV Compressor |
| Copper Wire | Copper Ingot → draw | LV Wiremill |
| Iron Plate | Iron Ingot → compress | LV Compressor |
| Basic Circuit | Copper Wire + Iron Plate + Redstone | LV Assembler |
| Aluminium Ingot | Bauxite Ore → smelt + process | LV Blast Furnace *(unlock trigger)* |

### Resource Nodes Unlocked at LV
- Iron Ore
- Redstone
- Bauxite Ore *(required for Aluminium)*

### Machines
| Machine | Function | EU/t |
|---|---|---|
| LV Generator | Coal/Charcoal → 8 EU/t | Produces EU |
| LV Extractor | Faster node extraction | 2 EU/t |
| LV Furnace | Ore/Dust → Ingot | 4 EU/t |
| LV Macerator | Ore → 2× Dust | 4 EU/t |
| LV Alloy Smelter | 2× inputs → Alloy | 6 EU/t |
| LV Compressor | Ingot → Plate | 4 EU/t |
| LV Wiremill | Ingot → Wire | 4 EU/t |
| LV Assembler | Multiple → Component | 8 EU/t |
| LV Blast Furnace | High-temp smelting (Aluminium, Steel) | 16 EU/t |
| LV Electrolyzer | Compound → Elements | 10 EU/t |

### Recipe Depth
**3–5 steps** — e.g. Ore → Macerate → Dust → Smelt → Ingot → Compress → Plate → Assemble

### Tier Unlock Trigger
Produce **1× Aluminium Ingot** in the LV Blast Furnace. This unlocks the MV tier research branch.

---

## Tier 2 — MV (Medium Voltage)

**Unlock condition:** Produce Aluminium in the LV Blast Furnace.

**Theme:** Industrial Scaling. Machines run faster and handle multi-step chemical processes. The player must upgrade their power network to MV cables. Stainless Steel — a complex multi-input alloy — is the milestone that closes this tier.

### Key Materials
| Material | Recipe | Machine |
|---|---|---|
| Aluminium Plate | Aluminium Ingot → compress | MV Compressor |
| Aluminium Wire | Aluminium Ingot → draw | MV Wiremill |
| Advanced Circuit | Aluminium Wire + Basic Circuit + Gold Foil + Electrum | MV Assembler |
| Chrome Dust | Ruby/Chrome Ore → macerate + electrolyze | MV Electrolyzer |
| Nickel Ingot | Nickel Ore → smelt | MV Furnace |
| Manganese Dust | Pyrolusite Ore → macerate | MV Macerator |
| Stainless Steel | Steel + Chrome Dust + Nickel + Manganese (6:1:1:1) → alloy | MV Alloy Smelter *(unlock trigger)* |

### Resource Nodes Unlocked at MV
- Gold Ore
- Nickel Ore
- Chrome / Ruby Ore
- Pyrolusite (Manganese) Ore

### Machines
| Machine | Function | EU/t |
|---|---|---|
| MV Transformer | Steps up LV → MV voltage | — |
| MV Generator | Better fuel → 32 EU/t | Produces EU |
| MV Furnace | Faster smelting | 8 EU/t |
| MV Macerator | Faster ore doubling | 8 EU/t |
| MV Alloy Smelter | Multi-input alloys | 16 EU/t |
| MV Compressor | Faster plates | 8 EU/t |
| MV Wiremill | Faster wire drawing | 8 EU/t |
| MV Assembler | Complex multi-input assembly | 16 EU/t |
| MV Electrolyzer | Chemical separation | 20 EU/t |
| MV Chemical Reactor | Chemical synthesis | 24 EU/t |
| MV Blast Furnace | High-temp alloys | 32 EU/t |

### Recipe Depth
**4–6 steps** — e.g. Chrome Ore → Macerate → Electrolyze → Dust → Alloy Smelter (with 3 other inputs) → Stainless Steel

### Tier Unlock Trigger
Produce **1× Stainless Steel Ingot** in the MV Alloy Smelter. This unlocks the HV tier research branch.

## Tier 3 — HV (High Voltage) — Final Tier (v1.0)

**Unlock condition:** Produce Stainless Steel in the MV Alloy Smelter.

**Theme:** Automation. The player has access to the full machine roster. The endgame challenge is building efficient, self-sustaining production lines capable of feeding the **Applied Energistics** network — a unified item storage and auto-crafting system. The goal of HV is not just to produce items but to automate the entire factory so that complex items craft themselves on demand.

### Key Materials
| Material | Notes |
|---|---|
| Stainless Steel | Carry-over from MV, used heavily in HV casings |
| Titanium | New HV metal — extremely complex chain |
| Iridium | Rare, end-game material |
| HV Circuits | Most complex circuit chain in the game |
| AE Storage Cell | Crafted from HV Circuits + Iridium + Stainless Steel casing — unlocks AE |

### Systems Unlocked at HV
- **Applied Energistics Network**: A unified wireless item storage and retrieval system. All machines can input/output to the AE network instead of directly to pipes. The player can request items to be auto-crafted on demand.
- **HV Power Network**: Requires HV cables and Transformer upgrades. Significantly more EU/t available.
- **Multi-block Machines** *(if implemented)*: Large structures that require multiple tile blocks to build, producing items impossible to craft in single-tile machines.

### Recipe Depth
**6–8 steps** — chains span multiple machines across all previous tiers

### Endgame Goal
Build and activate the **Applied Energistics Controller** — a multi-tile structure requiring HV Circuits, Stainless Steel Plates, Iridium Plates, and an AE Storage Cell. Activating it completes the game (v1.0 victory condition).

## Design Notes

- **Tier gates are production-based, not time-based.** No timers. The player progresses when they build the right factory, not when they wait long enough.
- **Each tier carries forward all previous machines.** Steam machines remain useful in HV — they are slow but cheap to run.
- **The Brick Furnace is intentionally primitive.** Its slowness is a feature — it creates the pressure to reach LV quickly.
- **Applied Energistics is a reward, not a requirement.** The player can complete HV goals without building the full AE network, but AE makes the final crafting chains tractable.
